# News Blog
SC153/5025/2024
Gichunge Samuel Mwangi

A modern, responsive news blog website showcasing articles with an elegant card-based layout and professional styling.

## Features

- **Responsive Grid Layout** - Articles displayed in a flexible card grid that adapts to different screen sizes
- **Card Design** - Each article is presented in a styled card with:
  - Gradient background (orange to yellow)
  - Box shadows for depth
  - Hover effects (scale animation)
  - Figure captions for images
  
- **Styled Navigation Bar** - Modern sticky navbar with:
  - Centered navigation links
  - Hover animations with background highlight and lift effect
  - Box shadow for visual separation
  
- **Logo Header** - Circular logo image in the header with shadow styling

- **Professional Typography** - Consistent font styling with Arial throughout

## Project Structure

```
websii/
```

## File Descriptions

### index.html
Contains the structure of the website including:
- Header with logo
- Navigation menu
- 20 article cards with figures and captions
- Footer with copyright information

### styles.css
Defines all visual styling:
- Body and general layout
- Header with radial gradient background
- Navigation bar with sticky positioning
- Card layout with flexbox
- Footer styling
- Figure and figcaption styling

### images/
Stores all images for:
- Logo (rao.jpg.jpg)
- 20 article images

## Getting Started

1. Open `index.html` in a web browser
2. Navigate through articles using the navigation menu
3. Hover over cards to see the scale animation effect
4. Hover over navigation links to see the interactive effects

- **Responsive Design**: Open the page in different screen sizes to see the flexible card layout
